## Overview
Follow the instructions below in sequential order to start using PLuG! This process should be quick and easy, taking no more than 10 minutes.



<TOCInline toc={props.toc} exclude="Overview" toHeading={2} />

## Create your DevOrg
1. Navigate to [app.devrev.ai](https://app.devrev.ai)
2. Login, and create a new DevOrg



![Button](/static/images/createorg.png)

3. Follow instructions and add necessary details

## Accessing DevOrg Credentials
1. Navigate to `DevOrg Settings`

<Image alt="settings" src="/static/images/settings.png" width={1824} height={1754} />
2. Go to `Support` 
   The page should contain the following:

<Image alt="support" src="/static/images/supportwindow.png" width={1921} height={1149} />

3. Generate a `Token` using a name and expiration date that suits you. Also for yourself maintain a `Widget Link` which should follow the form of https:://plug.devrev.ai/`{Your DevOrg Name}`/widget.js

## Post Credential Access

For the following two steps, make sure to reference the repository this website is on, [here](https://github.com/devrev/plg-starter) is a link.


## Widget Installation

1. Navigate to `components/widget.js`
2. Change the source of the script from the current link to your `Widget Link`

```js:components/widget.js {3} showLineNumbers
<Script
           id="devrev_plug"
           onLoad={async () => {
             try {
               const sessionToken = token
               const sdkObject = new window.DevRevSupportSdk({
                 sessionToken,
               });
               setSupportSdk(sdkObject);
             } catch (error) {
               console.log(error);
             }
           }}
           s̶r̶c̶=̶{̶"̶h̶t̶t̶p̶s̶:̶/̶/̶p̶l̶u̶g̶.̶d̶e̶v̶r̶e̶v̶.̶a̶i̶/̶n̶e̶x̶t̶p̶l̶u̶g̶/̶w̶i̶d̶g̶e̶t̶.̶j̶s̶"̶}̶
           // modify the source
           src={Enter your own link}
 />
```
